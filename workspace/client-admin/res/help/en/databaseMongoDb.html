A data source component is the function that allows the administrator to handle MongoDb data source (create, edit, delete, enable and disable a data source). This data source provides data that will be used by services (see dataset component). The following sections explain how to handle a database as a data source.

Create a MongoDb database

  1. Click on Create.
  2. Fill in the form to create a new database.
  3. You might click on Test Connection to verify that it works.
  4. Click on OK to validate.

Edit a database

  1. Select a database.
  2. Click on Edit.
  3. Use the form to modify entries.
  4. You might click on Test Connection to check.
  5. Click on OK to validate.

Delete a database

  1. Select a database.
  2. Click on Delete.

Enable a database

  1. Select a database.
  2. Click on Enable.

Disable a database

  1. Select a database.
  2. Click on Disable.

Testing connection

  1. Select a database.
  2. Click on Test Connection.
